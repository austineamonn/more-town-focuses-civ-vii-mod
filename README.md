# More Town Focuses Civ VII Mod

Author: austineamonn

Version beta

## Current Mod:

ex, distant lands w/ treasure resource only - gives 1 treasure fleet, +2 gold on treasure resources
ex, distant lands conquered - gives +3 treasure fleets
all, conquered settlements only - (-5 happiness per age) +50% rural yields except happiness
all, adjacent to nav river only - +2,+3,+4 food each age on river tiles
all, adjacent to lake - +2,+3,+4 food, +1,+1,+2 culture each age on lake tiles
all, on biome - +2,+3,+4 of yield on vegetated each age on all biome tiles (+1,+1,+2 for science/culture)
tundra - culture, grassland - food, desert - gold, plains - production, tropical - science

bonus for unique improvements

fort towns - gain +10% towards purchasing military units for all settlements
religious site - +2 slots
factory town - +100% towards purchasing train station and port

nerf the biome ones by requiring 3 settlements in said biome for unlock

## Latest Update:

Resolved issues with distant lands focuses being available in the homelands. Changed fort town bonus to work for all settlements (since it cannot just work for one).
