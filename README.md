# More Town Focuses Civ VII Mod

Author: austineamonn

Version beta

## Current Mod:

ex, distant lands w/ treasure resource only - gives 1 treasure fleet, +2 gold on treasure resources
ex, distant lands conquered - gives +3 treasure fleets
all, conquered settlements only - (-5 happiness per age) double rural yields except happiness
all, adjacent to nav river only - +2 food per age on river tiles
all, adjacent to lake - +3 food, +1 culture per age on lake tiles
all, on biome - +1 of yield on vegetated per age on all biome tiles (doubled for not science/culture)
tundra - culture, grassland - food, desert - gold, plains - production, tropical - science

fort towns - gain +25% towards purchasing military units
religious site - +2 slots
factory town - +100% towards purchasing train station and port

added town specializations.
